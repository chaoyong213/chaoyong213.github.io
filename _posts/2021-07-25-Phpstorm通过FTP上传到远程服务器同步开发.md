### Phpstorm通过FTP上传到远程服务器同步开发

配置
- [x] 找到 Tools > Deployment > Configuration
- [x] 点击新建一个FTP
- [x] 输入名称
- [x] 输入FTP信息 (主机地址、fpt用户名、密码)
- [x] 点击测试链接
- [x] 设置Mappings (选择文件地址)
- [x] 启用自动上传 (路径：Tools > Deployment > Automatic Upload(always))
- [x] 没了
